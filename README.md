# dotfiles

Setup `zsh` using [Oh My Zsh](https://ohmyz.sh/) and put config files for `git`.

## Prerequisites

- macOS with [Xcode Command Line Tools](https://developer.apple.com/xcode/) for Git
- [Homebrew](https://brew.sh/)
- `curl`

## Setup

```sh
./setup.sh
```

This will install and configure:

### Programming Languages

- [Go](https://golang.org/) via Homebrew
- [Python](https://www.python.org/) via [uv](https://docs.astral.sh/uv/) (fast Python package and project manager)
- [Node.js](https://nodejs.org/) via [nvm](https://github.com/nvm-sh/nvm)

### Developer Tools

- [VS Code](https://code.visualstudio.com/) via Homebrew Cask
- [Docker Desktop](https://www.docker.com/products/docker-desktop/) via Homebrew Cask
- [GitHub Copilot CLI](https://githubnext.com/projects/copilot-cli) via `gh` extension
- Git via Xcode Command Line Tools
